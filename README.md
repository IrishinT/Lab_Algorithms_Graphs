**Вариант 10** | **Тверской государственный технический университет** | **2026**

---

## Описание проекта

Данный проект представляет собой программу для анализа **сети поставок товаров** с использованием графовых алгоритмов. Граф моделирует логистическую инфраструктуру:

| Элемент графа | Предметная область | Пример |
|---------------|-------------------|--------|
| **Вершины** | Склады, магазины, распределительные центры | `Склад_Центральный`, `Магазин_1` |
| **Рёбра** | Маршруты доставки между объектами | `Склад_А → Магазин_1` |
| **Веса рёбер** | Стоимость доставки (тыс. руб.) | `15.0`, `25.5` |

---

## Сеть поставок товаров

| Параметр | Значение |
|----------|----------|
| **Предметная область** | Логистика и управление цепями поставок |
| **Вершины** | Склады и магазины (≥15) |
| **Рёбра** | Маршруты доставки (≥20) |
| **Вес ребра** | Стоимость доставки по маршруту |
| **Основная задача** | Найти оптимальный маршрут доставки товара |

## Реализованные алгоритмы

| Алгоритм | Класс | Методы | Сложность |
|----------|-------|--------|-----------|
| **BFS** (обход в ширину) | `BFS_Algorithm` | `Traverse()`, `IsReachable()`, `FindShortestPath()` | O(V + E) |
| **DFS** (обход в глубину) | `DFS_Algorithm` | `Traverse()`, `TraverseRecursive()` | O(V + E) |
| **Компоненты связности** | `Connectivity` | `FindConnectedComponents()`, `IsConnected()` | O(V + E) |

## 📝 Примеры использования

### 1. Загрузка графа

1. Укажите путь к файлу: `data/supply_network.txt`
2. Нажмите **«Обзор»** или введите путь вручную
3. Нажмите **«Загрузить»**
4. В поле вывода появится статистика:
   ```
   Граф успешно загружен!
   Вершин: 17
   Рёбер: 22
   Тип: Неориентированный
   ```

### 2. Обход BFS

1. Выберите стартовую вершину (например, `Склад_Центральный`)
2. Нажмите **«BFS»**
3. Результат:
   ```
   [BFS] Обход от объекта: Склад_Центральный
   Порядок посещения:
   Склад_Центральный → Склад_Север → Склад_Юг → Распределитель_А → ...
   
   Всего вершин: 17
   ```

### 3. Проверка достижимости

1. Выберите «От»: `Склад_Центральный`
2. Выберите «До»: `Магазин_1`
3. Нажмите **«Проверить путь»**
4. Результат:
   ```
   [Достижимость] Склад_Центральный → Магазин_1
   
   ✓ Объект Магазин_1 ДОСТИЖИМ из Склад_Центральный
   
   Кратчайший путь (2 перехода):
   Склад_Центральный → Склад_Север → Магазин_1
   ```

### 4. Компоненты связности

1. Нажмите **«Компоненты»**
2. Результат:
   ```
   [Компоненты связности]
   
   ✓ Сеть поставок СВЯЗНАЯ
   Все 17 объектов входят в единую сеть
   ```
