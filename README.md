# 📦 Система анализа сети поставок (Графы)
**Лабораторные работы №4–6 по дисциплине «Алгоритмизация и программирование»**

Данный проект представляет собой программный комплекс на языке **C# (WinForms)** для моделирования и глубокого анализа логистической сети поставок товаров. Система позволяет работать с объектами (склады, магазины) и связями между ними (маршруты доставки) с использованием теории графов.

---

## 🚀 Основные возможности

### Лабораторная работа №4: Построение и обход графа
*   **Хранение данных:** Граф реализован в виде **списка смежности**, что обеспечивает эффективную работу с памятью $O(V + E)$.
*   **Обход в ширину (BFS):** Поиск кратчайшего пути по количеству пересадок/ребер.
*   **Обход в глубину (DFS):** Проверка связности и исследование структуры сети.
*   **Компоненты связности:** Определение изолированных групп складов, не имеющих сообщения с основной сетью.

### Лабораторная работа №5: Взвешенные графы и оптимизация
*   **Алгоритм Дейкстры:** Поиск кратчайшего пути между двумя точками с учетом стоимости или времени доставки (веса ребер).
*   **Матрица расстояний:** Программный расчет минимальной стоимости доставки от выбранного склада до **всех** остальных объектов сети.
*   **Интерфейс:** Результаты выводятся в интерактивную таблицу `DataGridView` для удобства логиста.

### Лабораторная работа №6: Глубокий анализ (Вариант 10)
*   **Критические узлы (Articulation Points):** Поиск складов, выход из строя которых приведет к разрыву сети и невозможности поставок.
*   **Минимальное остовное дерево (MST):** Построение оптимальной инфраструктуры дорог с использованием **алгоритма Прима**, позволяющей связать все точки с минимальными затратами.
*   **Задача варианта:** Поиск **оптимального логистического хаба** — точки старта, обеспечивающей минимальную суммарную стоимость доставки по всей сети.

## 📊 Формат входных данных
Программа загружает описание сети из текстовых файлов `.txt`. 
Пример структуры файла (`supply_network.txt`):
```text
[SECTION:VERTICES]
Склад_А
Склад_Б
Магазин_Центральный

[SECTION:EDGES]
Склад_А, Склад_Б, 12.5
Склад_Б, Магазин_Центральный, 8.0
```

## 💻 Как запустить
1. Склонируйте репозиторий.
2. Откройте файл решения `.sln` в Visual Studio.
3. Соберите проект (Build).
4. Запустите приложение.
5. Нажмите кнопку **«Обзор...»** и выберите файл с графом из папки `data/`.
6. Используйте кнопки на панели **«Анализ»** для выполнения расчетов.
